**Guia Rápido de Uso – Sistema de Designação de Gestores Escolares**

**Versão:** 1.0  
**Objetivo:** Orientar o usuário no fluxo completo para realizar uma designação de diretores, vice-diretores e coordenadores conforme a Resolução SEDUC nº 162/2025.

---

### Fluxo Completo (passo a passo)

Siga esta ordem para garantir que o processo funcione corretamente:

#### 1. Cadastro de Escolas
1. Acesse **Escolas** no menu lateral.
2. Clique em **Nova Escola**.
3. Preencha:
   - Código INEP
   - Nome
   - Número de turnos
   - Tipo de merenda e limpeza
   - Modalidades (marque as checkboxes – ex: Regular, PEI, CASA, etc.)
4. Se a escola tiver modalidade **PEI**, marque também as opções "PEI + Parcial" e/ou "PEI 9h" quando aplicável.
5. Salve.

#### 2. Cadastro de Agentes Escolares
1. Acesse **Agentes Escolares**.
2. Clique em **Novo Agente**.
3. Preencha:
   - Nome completo
   - Matrícula funcional
   - Status (Ativo / Inativo / Licença)
4. Salve.

> **Dica:** Cadastre todos os agentes antes de iniciar os cálculos.

#### 3. Cadastro de Período de Processamento
1. Acesse **Períodos de Processamento**.
2. Clique em **Novo Período**.
3. Defina:
   - Data de início
   - Data de fim
4. Salve.
5. **Importante:** Apenas um período pode estar ativo por vez. O sistema ativa automaticamente o que a data atual (horário do servidor) está dentro do intervalo.

#### 4. Cálculo do Módulo
1. Acesse **Cálculo do Módulo** (página principal do fluxo).
2. Localize a escola desejada e clique em **Cálculo**.
3. Preencha:
   - Matrícula ativa
   - Professores dedicados (se tiver modalidade CASA ou Sistema Prisional)
   - Matrícula CEL (se tiver modalidade CEL)
4. Clique em **Calcular Módulo** (ou **Recalcular Módulo** se já existir cálculo).
5. Verifique os cargos calculados e a explicação dos critérios.
6. Clique em **Salvar Cálculo**.

> Após salvar, a escola passa a ter status **processado** e fica disponível para designação.

#### 5. Designação de Gestores
1. Ao salvar o cálculo, você é redirecionado a página de designação. Essa página também está disponível de Cálculo do Módulo, botão **Designar Gestores**.
2. Na página de designação:
   - Veja os cargos necessários.
   - Para cada cargo, na coluna **Nova designação**, clique no botão **+ Adicionar** e selecione os agentes desejados (busca com autocomplete).
   - Para cessar um agente anterior: clique no botão “Cessar” ao lado dele.
   - Na caixa **Contratação de terceiros**, selecione o status de contratação de **AOE** e **ASE**.
3. Clique em **Salvar Designação**.

---

### Dicas Gerais

- **Status da Designação** é calculado automaticamente:
  - **Pendente** → nada foi feito
  - **Em andamento** → designações parciais ou contratações incompletas
  - **Concluído** → todos os cargos preenchidos + AOE/ASE contratados

- Sempre verifique o **período ativo** no Dashboard ou no topo das páginas.

- O Dashboard mostra um resumo visual do andamento (total de escolas, módulos calculados, pendentes, etc.).
